# Site de recomendação de animes

Este projeto é uma plataforma web desenvolvida para ajudar os usuários a encontrar animes com base em suas preferências.
Utilizando algoritmos de recomendação, o site sugere orientações de acordo com o histórico de avaliações, gêneros favoritos
e tendências populares.

🛠️ Tecnologias Utilizadas
HTML5 para estruturação do conteúdo
CSS3 para estilização e responsividade
